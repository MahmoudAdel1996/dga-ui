Usage Notes

  • Placement: The topbar sits above the main <header> / <nav> and below no other element. It has the highest z-index in the layout.
  • Collapse: Uses Bootstrap's data-bs-toggle="collapse" — no JavaScript required beyond the Bootstrap bundle.
  • DGA badge: The registration number and link are required for official government websites registered with the Digital Government Authority.
  • RTL: Add dir="rtl" on the topbar wrapper when the page language is Arabic. Bootstrap's RTL stylesheet handles all layout mirroring.
  • Flag: The SASO app uses an SVG asset (assets/images/sa.svg). In this demo we use the emoji flag 🇸🇦 as a fallback.
