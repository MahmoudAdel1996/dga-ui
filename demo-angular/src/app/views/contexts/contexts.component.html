How to Use

Available Context Classes:

  • .on-primary - White text on primary color
  • .on-dark - White text on dark background
  • .on-danger - White text on danger/error color
  • .on-success - White text on success color
  • .on-warning - Dark text on warning color
  • .on-light - Dark text on light background

What Adapts Automatically:

  • ✅ Buttons (all variants)
  • ✅ Links
  • ✅ Form inputs and selects
  • ✅ Text colors
  • ✅ Border colors
  • ✅ Cards and alerts
  • ✅ Tables and badges

Example Code:

<div class="on-primary p-4">
  <h3>Content adapts automatically</h3>
  <button class="btn btn-primary">Primary Button</button>
  <a href="#">Link</a>
  <input type="text" class="form-control" placeholder="Input">
</div>
