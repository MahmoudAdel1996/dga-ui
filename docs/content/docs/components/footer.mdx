## Key Features

- **Multi-column Navigation**: Organize footer links into logical groups (up to 5 columns)
- **Social Integration**: Built-in support for social media buttons (Twitter, Facebook, Instagram, LinkedIn)
- **Accessibility Controls**: Dedicated section for accessibility tools (text size, read aloud, universal access)
- **Responsive Layout**: Adapts from multi-column on desktop to stacked on mobile
- **Two Color Variants**: Primary (`on-primary`) and Neutral (`bg-neutral-100`) backgrounds
- **Bottom Meta**: Copyright, legal links, and logo placement

<Callout type="info">
  Use `on-primary` for prominent footers or `bg-neutral-100` for subtle backgrounds. Bootstrap icons (`bi-*` classes) work great for social and accessibility buttons.
</Callout>
