# Tables

Display structured data with flexible table layouts including default, side-borderless, and striped variants. All tables include checkbox selection in the first column.

## Default Table

Standard table with checkboxes for row selection and linked data cells.

<Preview>
  <DefaultTable />
</Preview>

## Striped Table

Table with alternating row colors using the `table-striped` class. Improves readability by creating visual separation between rows.

<Callout type="info">
  Use `table-responsive` wrapper for mobile-friendly tables. Combine `table-striped` with `side-borderless` for a modern, clean appearance. Add `form-check-input-sm` to checkboxes for compact sizing.
</Callout>
