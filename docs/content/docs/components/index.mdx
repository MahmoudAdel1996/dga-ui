# Components Overview

Browse practical examples for the most common SDGA UI components. Each page includes ready-to-use HTML snippets that work out of the box with `sdga-ui/css/dga-ui.css`.

<Callout type="info">
  All examples assume you already imported SDGA UI CSS:

```html
<link rel="stylesheet" href="node_modules/sdga-ui/css/dga-ui.css" />
```

</Callout>

## Components

- **[Alerts](/docs/components/alerts)** — contextual, dismissible, and icon alerts
- **[Buttons](/docs/components/buttons)** — solid, outline, sizes, and icon buttons
- **[Cards](/docs/components/cards)** — headers, footers, media, and grids
- **[Forms](/docs/components/forms)** — inputs, floating labels, validation, switches
- **[Tables](/docs/components/tables)** — striped, bordered, hover, compact tables
- **[Footer](/docs/components/footer)** — navigation groups, social links, accessibility tools

Need another component? Open an issue on [GitHub](https://github.com/MahmoudAdel1996/dga-ui/issues) and we’ll add it.
