# Cards

Cards are flexible containers for grouping related content. SDGA UI cards support icons, checkboxes, footers, and responsive grids.

## Card with Icon

Cards can display an icon in the title area for visual identification.

<Preview>
  <CardWithIcon />
</Preview>

```html
<div class="card" tabindex="0">
  <div class="card-body">
    <h5 class="card-title">
      <i class="bi bi-check-circle card-title-icon"></i>
    </h5>
    <h6 class="card-subtitle mb-2">Card Title</h6>
    <p class="card-text">This is a description of the card content.</p>
    <div class="card-footer">
      <a href="#" class="btn btn-outline-secondary">Secondary</a>
      <a href="#" class="btn btn-primary">Primary</a>
    </div>
  </div>
</div>
```

## Card Grid

Combine cards in responsive grids using Bootstrap's grid system.

<Preview>
  <CardGrid />
</Preview>

```html
<div class="row row-cols-1 row-cols-md-3 g-3">
  <div class="col">
    <div class="card h-100" tabindex="0">
      <div class="card-body">
        <h5 class="card-title">
          <i class="bi bi-check-circle card-title-icon"></i>
        </h5>
        <h6 class="card-subtitle mb-2">Service 1</h6>
        <p class="card-text">First service description.</p>
        <div class="card-footer">
          <a href="#" class="btn btn-primary">Action</a>
        </div>
      </div>
    </div>
  </div>
  <!-- Repeat for additional cards -->
</div>
```

<Callout type="tip">
  Use Bootstrap icons (`bi-*` classes) in card titles and combine with `h-100` class on cards for equal heights in grids.
</Callout>
