# Forms

SDGA UI provides styled form controls with multiple states and sizes, including checkboxes, radios, and various visual states.

## Checkboxes

### Primary Checkboxes

Primary checkboxes with ripple effect in multiple sizes.

<Preview>
  <PrimaryCheckboxes />
</Preview>

```html
<div className="form-check form-check-md mb-2">
  <input type="checkbox" className="form-check-input ripple" id="chkPrimaryMd" checked />
  <label className="form-check-label" htmlFor="chkPrimaryMd">Medium</label>
</div>
<div className="form-check form-check-sm mb-2">
  <input type="checkbox" className="form-check-input ripple" id="chkPrimarySm" checked />
  <label className="form-check-label" htmlFor="chkPrimarySm">Small</label>
</div>
<div className="form-check form-check-xs mb-2">
  <input type="checkbox" className="form-check-input ripple" id="chkPrimaryXs" checked />
  <label className="form-check-label" htmlFor="chkPrimaryXs">Extra small</label>
</div>
```

</Callout>
