# Interactive Preview Examples

The Preview component supports two modes:

1. **Static Preview** (default) - Simple iframe preview
2. **Interactive Preview** - WebContainer-powered preview with code editing capabilities

## Static Preview (Default)

Standard preview with just HTML rendering:

## Usage

### Basic Static Preview

```jsx
<Preview>
  <button className="btn btn-primary">Click me</button>
</Preview>
```

### Interactive Preview with Custom Code

```jsx
<Preview
  interactive
  code={`
  <div class="alert alert-success">
    <h4>Success!</h4>
    <p>This is an interactive preview.</p>
  </div>
`}
></Preview>
```

## Features

- **Tab Navigation**: Switch between preview and code view
- **WebContainer Integration**: Runs real HTML/CSS/JS in an isolated environment
- **Open in New Window**: Click the â†—ï¸ button to open preview in a new tab
- **Arabic Support**: Full RTL support with Arabic UI
- **Loading States**: Shows loading spinner while WebContainer boots
- **Fallback**: Falls back to static preview if WebContainer fails
