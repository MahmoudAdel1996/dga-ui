## Features

- **Responsive Design**: Collapses to hamburger menu on mobile devices
- **Icon Support**: Add Bootstrap Icons to navigation links using `margin-inline-end` for RTL support
- **Active States**: Use `.active` class on links to indicate current page
- **Disabled Links**: Use `.disabled` class for non-interactive links
- **Mega Menu Dropdowns**: Full-width dropdown menus with multiple columns
- **Dropdown Groups**: Use `.dropdown-item-group` to create organized sections

## Link States

Navigation links support multiple states:

- **Default**: Standard link appearance
- **Hover**: `background-color: $neutral-100` with indicator
- **Active**: `background-color: $primary-600` with white text
- **Disabled**: Grayed out with `pointer-events: none`

<Callout type="tip">
  Use `margin-inline-end` for icon spacing in navigation links to ensure proper RTL (right-to-left) language support.
</Callout>

## Mega Menu Structure

The dropdown mega menu uses a flex layout with multiple columns:

- Each column is a `.dropdown-item-group`
- Headers use `.dropdown-header` with primary color
- Links use `.link-neutral` class with small size
- Full-width layout with custom padding (`padding: 2rem 5rem`)

<Callout type="info">
  The navbar dropdown requires Bootstrap JavaScript for toggle functionality. Ensure Bootstrap JS is included in your project.
</Callout>
